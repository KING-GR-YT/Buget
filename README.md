A **modern personal financial management web application** that helps you manage **expenses, investments, and financial growth** in one place.

This project is designed for **simple financial tracking with real-time Firebase data synchronization**.

---

# 📊 Features

## 🧾 Expense Management

* Add daily expenses
* Categorize spending
* Automatically update totals
* Real-time data sync with Firebase

## 📈 Investment Tracker

* Track **Fixed Deposits (FD)**
* Calculate **interest earned till date**
* Automatic **maturity date calculation**
* Daily interest updates

## 📊 Financial Dashboard

* Overview of total finances
* Quick financial summary
* Clean data tables

## 🔐 Secure Login

* Firebase email authentication
* Remember session option
* Login using **Enter key**

## ☁️ Cloud Database

* Firebase Firestore integration
* Live updates without page refresh
* Secure data storage

## 📱 Responsive UI

* Mobile friendly
* Futuristic dashboard design
* Clean and readable tables
